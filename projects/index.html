LApi

Progress

LApi is Discord Bot API written in Java by a single Person. It is still Work in Progress: Not all Gateway events are accessible yet. Some more caching needs to be done too. And a lot of HTTP requests are still missing. But the basic architecture is set up.

It is based on the Discord Docs , but on a lot of extra functionality is and will be worked on.

A basic documentation on how to get started can be found here.

Installation

In order to add LApi to your Project, add the following to your build.gradle file:
repositories {
    mavenCentral()
}
dependencies {
    implementation 'io.github.lni-dev:lapi:[version]'
}

Replace [version] with the current version (see Github).

Data is a container to store key-value pairs. These will remain in the same order, they have been added to the Data object.
